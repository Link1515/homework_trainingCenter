# 桌遊網

## 發想

- 讓桌遊玩家有更好的遊戲體驗

## 網頁架構

### 前台

- 遊玩桌遊
  - 官方
  - 玩家自製
- 桌遊推薦排行
  - icon 連結到桌遊製作
- 註冊
- 登入

### 一般後台

- 用戶資訊(資料修改)
- 好友
  - 好友聊天
- 我的最愛
- 製作/上傳桌遊
- 錯誤回報

### 管理者後台

- 使用者管理
- 桌遊管理
- 用戶回饋
- 檢舉管理
